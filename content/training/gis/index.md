### Assiniboine River Project
Applied GIS tools to extract high precision surface elevation data to create surface elevation cross sections of the Assiniboine River to assess the degree of superelevation.
- Used QGIS to create a mosaic of LiDAR scans of southwestern Manitoba to create a regional DEM for the project
- Applied Inverse Distance Weighting to construct a REM of the mosaic from the elevation profile of the Assiniboine River using geo-processing tools
- Using the REM used colour ramps and hillshade to identify areas to gather cross sectional data from
- Extracted cross-sectional elevation data from the clean DEM using line shape files.
